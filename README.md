# FixOS for OpenComputers 2: Reimagine

Локальный прототип интерфейса FixOS в стиле Windows 8/10 для идеи ОС под `OpenComputers 2`.

Что уже есть:

- рабочий стол с иконками
- меню Пуск
- панель задач и часы
- контекстное меню
- проводник с виртуальной файловой системой
- параметры системы
- встроенный браузер
- установщик `installer/install_fixos.bat`
- отдельная Lua-заготовка для `OpenComputers 2` в папке `oc2`

Как запустить:

1. Открой `installer/install_fixos.bat` и установи проект в любую папку.
2. После установки запусти `Launch FixOS.bat`.

Или открой `app/index.html` напрямую в браузере.

Важно:

- это визуальный и UX-прототип
- для OC2 я уже добавил стартовую Lua-оболочку и установщик в [oc2/README.md](D:/FixOSOP2/oc2/README.md)
